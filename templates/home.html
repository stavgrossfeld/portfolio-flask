Craigslist Crime Analysis

Using data from archived craigslist posts and reported crimes, performed a comprehensive analysis of reported crime trends and rent hikes in different neighborhoods of San Francisco. Presented information with exploratory graphs and predicted prices of housing using districts, years, bedrooms, and aggregate crime counts per district per year. Created heat-map of pricing with dynamic clustering of user inputted crimes. Possible uses of this project is a web application using this map style indicating crimes and rental prices from craigslist in real-time.
